## 1. Sebutkan dua karakteristik class atau object!

Class biasanya diawali dengan kata "class" saat inisiasi nya. Seperti


Sedangkan Object, ditandai dengan "new" saat inisiasi. Jadi formatnya adalah Seperti
